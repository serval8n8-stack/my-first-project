# Реализованные функции проекта МамаЭксперт

## Что было сделано

### 1. Сайт-эксперт детских товаров

**Целевая аудитория:** Мамы с детьми от 0 до 5 лет

**Визуальный стиль:** Мягкий пастельный дизайн с градиентами розового, сиреневого и мятного цветов.

---

### 2. Блог со статьями

Раздел с полезными статьями для мам:
- Как выбрать первую коляску для новорождённого
- Лучшие развивающие игрушки для малышей до года
- Переход на общий стол: советы педиатра
- Приучение к горшку: мягкий подход
- Подготовка к детскому саду: что нужно знать
- Обзор детских бутылочек: какую выбрать

---

### 3. Фильтрация по возрасту ребёнка

Интерактивные кнопки для фильтрации контента:
- Все возрасты
- 0–6 месяцев
- 6–12 месяцев
- 1–2 года
- 2–3 года
- 3–5 лет

При нажатии на кнопку отображаются только статьи для выбранной возрастной группы.

---

---

### 5. Реферальные ссылки

На каждой карточке бренда добавлены кнопки покупки с партнёрскими ссылками:

- **Ozon** (синяя кнопка)
- **Wildberries** (фиолетовая кнопка)
- **Яндекс Маркет** (жёлтая кнопка)

Ссылки открываются в новой вкладке. Сейчас используются заглушки `YOUR_REF_ID_*`, которые нужно заменить на реальные партнёрские ID.

---

### 6. Форма подписки

Секция для сбора email-адресов посетителей с возможностью подписки на еженедельную рассылку.

---

### 7. Адаптивная вёрстка

Сайт корректно отображается на мобильных устройствах:
- Шапка становится вертикальной
- Карточки перестраиваются в одну колонку
- Кнопки фильтров уменьшаются

---

## Технические детали

- Один HTML-файл со встроенными CSS и JavaScript
- Без внешних зависимостей (библиотек, фреймворков)
- Фиксированная шапка при прокрутке
- Плавные анимации при наведении
